# Roadmap

Planned milestones and priorities for AI-OS development.

## Current phase

**Phase 0 — Foundation** — Repository structure, documentation, configuration templates, and contribution guidelines. No application runtime yet.

## Upcoming phases

| Phase | Focus |
|-------|--------|
| 1 — Knowledge | Ingestion, processing, indexing |
| 2 — Memory | Persistent context, retention, privacy |
| 3 — Decision engine | Prompts, templates, evaluation |
| 4 — Integration | Orchestration, CLI/API |
| 5 — Hardening | Tests, CI, observability, security |

See [../architecture/system-overview.md](../architecture/system-overview.md) for phase details.

## How to propose roadmap changes

1. Open an issue or discussion describing the goal and phase fit.
2. For large shifts, draft a short ADR in [../decisions/](../decisions/).
3. Update this directory with a dated milestone file when approved, e.g. `2026-Q3-knowledge-mvp.md`.
